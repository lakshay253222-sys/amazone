## 🔍 Overview

A clean and responsive **Amazon Careers demo page** that replicates the look and feel of a professional job application portal. Users can browse why they should join, explore employee benefits, and submit a role application — all in a single polished page.

---

## ✨ Features

- 📋 **Employee Registration Form** — Select a role and submit an application with form validation
- 🏢 **Why Join Amazon?** — Career growth, innovative work, and great culture sections
- 🎁 **Employee Benefits** — Health support, flexible work, and learning opportunities
- ✅ **Form Success Message** — "Thank you" confirmation on successful submission
- 🔄 **Reset Button** — Clear the form in one click
- 📱 **Responsive Design** — Works on all screen sizes
